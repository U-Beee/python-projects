# My Python Projects

A collection of beginner Python projects I built while learning programming fundamentals.

---

## 1. Tic-Tac-Toe (`tic_tac_toe.py`)

A two-player Tic-Tac-Toe game that runs in the terminal, where players take turns choosing a position (1–9) on a 3×3 grid. The program automatically checks for a winner or a draw after every move.

**How to run:** `python tic_tac_toe.py`

---

## 2. Hangman (`hangman.py`)

A word-guessing game where the player tries to figure out a hidden word by suggesting one letter at a time. The player has 6 wrong guesses before the hangman is complete and the game is lost.

**How to run:** `python hangman.py`

---

## 3. Snake (`snake.py`)

A graphical Snake game built with Python's built-in `turtle` library, where the player steers a growing snake to eat red food while avoiding the walls and the snake's own body. Each piece of food eaten adds a segment to the snake and increases the score.

**How to run:** `python snake.py` *(requires no extra installation — `turtle` is built into Python)*

---

## 4. Pong (`pong.py`)

A two-player Pong game built with the `turtle` library, where each player controls a paddle to keep the ball from passing their side. The first player to reach 5 points wins.

**Controls:**
- Player 1 (left): `W` = up, `S` = down
- Player 2 (right): `↑` = up, `↓` = down

**How to run:** `python pong.py`

---

## 5. Typing Test (`typing_test.py`)

A terminal-based typing speed test that displays a random sentence and measures how fast (in Words Per Minute) and how accurately the player types it. Results are shown at the end with a rating.

**How to run:** `python typing_test.py`

---

## What I Learned

Through these projects I practised:
- Variables, loops (`while`, `for`), and `if/else` conditions
- Functions and how to organise code into reusable pieces
- Lists and basic data structures
- Working with Python's `turtle`, `random`, and `time` libraries
- Debugging and thinking step by step when solving problems
